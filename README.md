# Z80MiniProgect

Всім привіт, сьогодні вирігив поділитися своїм знаннями на Z80, своєю платою та схемою.
Нажаль код на асм писав тоді коли ще не було оперативки, тому код просто написаний без оперативки лише з використанням постійної пам'яті. Я реалізував інцілізацію дисплея LCD1602A, та обробки кнопок через мікросхему 74HC244N, в коді реалізував лише обробку однієї кнопки та виводу її стану на дисплей. Але схема абсолютно працездатна та може використовувати оперативку.
всі фото проекту можна переглянути у піпці img. А у папці code ви знайдете цей самий код на asm. В папці Gerber знаходяться усі файли для виготовлення плати.
Список компонентів які використовувалися:
CPU Z80
ROM 27C512
RAM AS6C4008
Дешифратори 74HC138
мікросхема утримання стану сигналу для дисплея HC74HC573
мікросхема використана лоя обробки кнопок 74HC244N
мікросхема скидання RSP MCP130, можна використовувати будь яку інншу мікросхему скидання
кварцовий генератор DIP14 2MHz
лінійний стабілізатор напруги 7805 5V
6 резисторів підтяжки кнопок до GND, опір 10k
4 резистори для підтяжки сигналів Z80 WAIT, NMI, INT, BUSREQ, опір 5.6k
1 резистор для світлодіода на 330 ом
9 конденсаторів на 100nf для стабілізації живлення мікросхем 
1 конденсатор 100mkf
1 світлодіод 
1 роз'єм Jack
